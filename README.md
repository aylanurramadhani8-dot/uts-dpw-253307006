Deskripsi
Proyek ini dibangun dengan sistem Multi-Page yang terdiri dari tiga halaman utama, yaitu index.html sebagai beranda, alur.html untuk informasi langkah pendaftaran, dan form.html untuk pengisian data. Agar proyek tetap bersih dan rapi (clean project), seluruh aset pendukung dipisahkan ke dalam folder asset/, di mana file CSS disimpan dalam subfolder css, file JavaScript dalam javascript, dan foto atau ilustrasi dalam folder img. Seluruh halaman ini disusun menggunakan elemen HTML5 Semantik seperti <header>, <nav>, <main>, dan <footer> untuk memastikan struktur kode yang standar dan mudah dibaca oleh sistem.
Pada halaman Beranda, pengunjung disambut dengan informasi formal serta tabel jadwal pelaksanaan PPDB yang dibuat rapi menggunakan tag <table>, <thead>, dan <tbody>. Halaman Alur berfungsi untuk memberikan panduan visual kepada calon siswa dengan menampilkan gambar ilustrasi yang dipanggil langsung dari folder aset, serta dilengkapi daftar urutan nomor (<ol>) untuk menjelaskan prosedur pendaftaran. Sementara itu, halaman Formulir menyediakan tempat input data interaktif mulai dari nama lengkap, nomor NIK, hingga pilihan jalur pendaftaran melalui menu dropdown. Tombol "Kirim Pendaftaran" pada formulir ini didesain khusus menggunakan tipe button agar data tidak langsung terkirim sebelum melewati tahap validasi.
Tampilan website menggunakan tema biru profesional dengan sentuhan desain modern pada CSS3. Bagian atas website atau header didesain menggunakan gradasi dua warna (linear-gradient) untuk memberikan kesan mewah. Selain itu, terdapat efek animasi pada menu navigasi dan tombol yang akan sedikit membesar atau bergeser ketika disentuh kursor, membuat website terasa lebih hidup. Tabel jadwal juga diberikan fitur zebra striping agar baris data lebih mudah dibedakan, serta kotak input formulir yang akan mengeluarkan efek cahaya biru (glow) saat difokuskan atau sedang diketik.
Untuk fungsi pintar, website ini dilengkapi dengan file script.js yang mengatur jalannya validasi data dan notifikasi. Terdapat fitur Custom Toast Notification atau pesan melayang yang akan muncul di bagian bawah layar untuk memberi tahu apakah pendaftaran sukses atau gagal. Sistem ini secara otomatis akan memeriksa apakah ada kolom yang kosong dan memastikan NIK yang diinput harus berupa angka yang berjumlah tepat 16 digit. Jika seluruh data sudah benar, sistem akan menampilkan pesan sukses, mengirimkan notifikasi, dan secara otomatis mengosongkan kembali seluruh isi formulir agar bersih seperti semula.
